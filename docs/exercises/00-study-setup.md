# Exercise 0: Study Setup And Workflow

Linked module:

- [Module 0 · How To Use This Course](/modules/00-how-to-use-this-course)

Source tag: `Stable concept`

## Goal

Set up a study rhythm that makes the rest of the course sustainable.

## Deliverables

Produce:

1. one completed study-session note using the session template
2. one short statement of your current mental model of Effect
3. one list of questions you expect the course to answer

## Required Tasks

### Task 1: Fill the session template

Use [Session Template](/meta/session-template) and fill:

- current module
- time budget
- source files to read
- one concept you expect to find difficult

### Task 2: Define your baseline

Write 5 to 8 sentences on:

- what you think `Effect` is
- how you think it differs from `Promise`
- what you think services and layers might be

Do not research before writing this. The point is to make your current model
explicit.

### Task 3: Define your learning constraints

Write down:

- how many sessions per week you can realistically sustain
- whether you prefer reading before coding or coding before reading
- what kinds of examples help you learn fastest

## Completion Criteria

You are done when:

- you have a concrete session plan
- you have written down your starting assumptions
- you have a repeatable workflow for each module

## Review Checklist

- Did you make your current assumptions explicit?
- Did you keep the plan realistic?
- Do you know how you will use Codex during the course?

## Ask Codex

- "Review my Module 0 setup."
- "Tell me which of my assumptions are likely wrong."
- "Make my study plan stricter and more realistic."
